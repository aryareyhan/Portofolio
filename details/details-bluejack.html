Bluejack V2

  • Description:An improved iteration of Bluejack Pharmacy, an Android Java app aimed at streamlining the medicine purchasing and browsing process. It employs Volley to efficiently retrieve data from a JSON link and integrates images using Picasso. Users must verify their accounts with OTP codes from a messaging app. Furthermore, it ensures data integrity by utilizing SQLite to store user and transaction information, granting users easy access to their purchase history.
  • Technologies:Android Java, SQLite, Google Maps API, Volley, Picasso
